# Techfix V2 Premium

Site institucional premium da Techfix Soluções Residenciais e Comerciais.

Como subir no GitHub:
1. Extraia o ZIP.
2. No GitHub, abra o repositório techfix.
3. Clique em Add file > Upload files.
4. Envie index.html e styles.css.
5. Clique em Commit changes.
6. Aguarde o GitHub Pages atualizar.

Observação: troque o número do WhatsApp no index.html quando quiser usar o número real.
